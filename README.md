# Compiling

The prefered source is using `esy`.

## Using `esy`

```bash
# Installing esy
npm install esy
# Installing the dependencies and compiling
esy
```

Then, to access it on a webbrowser, type:
```bash
# Run a local server on port 8000.
esy server > /dev/null &
# Then open the browser.
firefox localhost:8000 &
```
